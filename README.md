##  Part1 - Intro to Federated Learning

联合学习允许在多个设备或组织中训练模型，而无需共享数据，从而提高隐私和安全性。联合学习还有许多实际用途，例如在移动键盘上训练下一个单词预测模型，而无需将敏感的击键传输到中央服务器。
